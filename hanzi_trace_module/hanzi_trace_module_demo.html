Hanzi Trace Module — Minimal Demo

Module shows only Undo, Clear All, and Save Image (with tooltips). It reads <char>_settings.json, <char>_strokes.json, and <char>_dots.json from the base folder.

Included sample files for testing: media/hanzi/家_settings.json, media/hanzi/家_strokes.json, media/hanzi/家_dots.json. Replace these with your exported ones.
